## Stedi Demo Mapping

## Summary

This Mapping shows how a `JEDI 850` can be mapped to a fictional API shape.

You can see how each value was mapped in this [15min video tutorial](https://www.youtube.com/watch?v=b0sPfOrPL3o&t=1s).
